# research

Источник идей для агента. Кидай сюда транскрипты постов и видео, которые тебе зашли — то, в чьих темах и стиле ты хочешь работать.

Формат простой: один файл = один материал. Внутри:

- ссылка-источник;
- сам транскрипт или текст;
- 1–2 строки, чем зацепило.

Стратег и писатель читают эту папку и достают отсюда идеи и разборы. Чем больше насмотренного тут лежит, тем точнее темы и тем меньше агент выдумывает из воздуха.

Пусто на старте — это нормально. Можно начать с тем из головы (`content/inbox.md`) и докидывать сюда по ходу.
