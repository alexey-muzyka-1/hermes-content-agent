# Воронка

Каждый пост должен знать, какой этап воронки он закрывает. Не знает — не пиши, сначала определи угол.

## Карта (пример, перепиши под свои платформы)

```text
Threads / X        (верх / awareness — охват, личный бренд)
   ↓
LinkedIn           (середина / credibility — доверие, B2B)
   ↓
Instagram / Reels  (лицо / distribution — короткое видео, разборы)
   ↓
Telegram           (низ / community — глубокий контекст, дневник)
   ↓
Продукт            (conversion — trial / demo / продажа)
```

## Роль платформ

| Платформа | Этап | Зачем | Частота |
|---|---|---|---|
| <Threads> | awareness | короткие ценностные посты, охват | <6x/день> |
| <LinkedIn> | credibility | кейсы, методологии, доверие | <2-3x/нед> |
| <Reels> | distribution | лицо, видео, разборы | <1x/день> |
| <Telegram> | community | дневник, контекст, эксклюзивы | <1-2x/день> |
| <Продукт> | conversion | кейсы, CTA, trial | по необходимости |

## Принцип
Один пост — один выход. Не складывать несколько CTA. Если задача — доверие, CTA может не быть вообще.
